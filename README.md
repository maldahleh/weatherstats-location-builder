# WeatherStats Location Service

This repo houses a Go microservice that scrapes Environment and Climate Change Canada's servers to build a mapping of
climate station IDs to consumable station names, along with providing information on what climate data is available
at each station.

### Endpoints
**/stations** - Returns a mapping of all climate stations in Canada
